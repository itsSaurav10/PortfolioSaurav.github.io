Competetive Programming

Let us start with the very basics of competetive programming. It is like any other mind sport where you will be given some tasks and you got to solve it in some fixed amount of time. Let me talk about CodeForces for instance. It is one of the world's leading competetive coding platforms with users all over the world. It holds contests weekly for both division 1 (users rated over 2100) and for division 2(users rated under 2100). There are educational rounds, ICPC style rounds.
